# Image Assets

Place your logo and favicon files in this folder with the following names:

## Required Files

### `logo.png`
- **Used for:** Login screen
- **Recommended size:** 300x300px or larger (will be displayed at ~96px height)
- **Format:** PNG with transparent background recommended
- **Purpose:** Main branding on login page

### `favicon.png`
- **Used for:**
  - Browser tab icon
  - Top-left navigation menu icon
- **Recommended size:** 32x32px, 64x64px, or 128x128px
- **Format:** PNG
- **Purpose:** Small icon that represents your app

## Fallback Behavior

If these files are not present, the app will automatically fall back to default icons:
- Login screen: Blue gradient icon with checkmark
- Navigation: Blue gradient box with package icon
- Browser tab: Default Vite icon

## How to Add Your Images

1. Save your logo as `logo.png` in this folder
2. Save your favicon as `favicon.png` in this folder
3. Refresh your browser - the images should appear automatically

No code changes needed!
